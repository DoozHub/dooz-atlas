# dooz-website-builder

> AI-powered website generation platform

## Overview

Dooz Website Builder enables no-code website creation with AI-assisted design and content generation. Build professional websites in minutes without coding expertise.

## Features

### AI-Powered Generation
- Describe your website, AI builds it
- Component suggestions based on industry
- Content generation for pages
- Image recommendations

### Visual Editor
- Drag-and-drop components
- Real-time preview
- Mobile responsive editing
- Component library

### Export Options
- Static site export (HTML/CSS/JS)
- Next.js export
- Vercel/Netlify deployment
- Custom domain support

## Tech Stack

- **Language**: TypeScript
- **Framework**: Next.js
- **AI**: Gemini/OpenAI integration
- **Styling**: TailwindCSS
