## Key Features

| Feature | Description |
|---------|-------------|
| **Visual Builder** | Drag-and-drop workflow editor |
| **AI Actions** | Natural language task descriptions |
| **Local Execution** | Run on your machine, not cloud |
| **Integrations** | 100+ app connectors |
| **Scheduling** | Cron-based and event triggers |
| **Monitoring** | Real-time execution logs |
