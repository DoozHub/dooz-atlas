# dooz_ballpark_ai

> AI-powered project estimation and quoting tool

## Overview

Ballpark AI helps generate accurate project estimates and quotes using machine learning trained on historical project data.

## Features

- **Requirements Analysis**: Parse requirements docs for estimation
- **Cost Calculation**: Automatic cost estimation engine
- **Learning System**: Improves accuracy from historical data
- **Quote Generation**: Professional quote documents

## Tech Stack

- **Language**: PHP
- **AI**: Gemini/OpenAI integration
- **Database**: Integration with dooz-core
