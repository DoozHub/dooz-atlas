## Overview

Dooz Hub is a local-first media management application for photographers, videographers, and content creators. Organize, tag, and manage your media library with AI-powered features while keeping files on your own storage.

## Key Features

| Feature | Description |
|---------|-------------|
| **Media Library** | Unified view of photos, videos, and documents |
| **Smart Albums** | AI-generated albums based on content |
| **Face Recognition** | Automatic face detection and grouping |
| **Geo-tagging** | Map view with location-based organization |
| **RAW Support** | Native support for camera RAW formats |
| **Non-Destructive** | Original files never modified |
| **Cloud Backup** | Optional encrypted backup via Dooz Sync |

## Supported Formats

### Images

### Documents
