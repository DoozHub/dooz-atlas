# worklog

> Detailed work logging and time tracking system

## Overview

Worklog provides comprehensive work logging with detailed notes, project-based tracking, and billable hours calculation.

## Features

- **Time Entries**: Log work with detailed notes
- **Project Tracking**: Organize by project/client
- **Billable Hours**: Mark entries as billable/non-billable
- **Export**: Generate invoices and reports

## Integration

- Syncs with dooz-core for central billing
- Calendar integration for scheduling
- Export to PDF/CSV

## Tech Stack

- **Language**: PHP (Blade)
- **Database**: Synced via dooz-core
