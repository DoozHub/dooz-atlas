# dooz-app-builder

> Low-code application builder for the Dooz ecosystem

## Overview

Dooz App Builder provides a visual interface for creating custom business applications that integrate with the Dooz platform.

## Features

### Visual Workflow Designer
- Drag-and-drop action blocks
- Conditional logic
- Loop constructs
- Integration with Dooz events

### Database Schema Builder
- Visual table designer
- Relationship mapping
- Migration generation
- Seed data support

### API Endpoint Generation
- RESTful endpoint creation
- GraphQL schema generation
- Authentication handling
- Rate limiting

### App Marketplace Publishing
- One-click publish to marketplace
- Version management
- Pricing configuration
- Review process

## Tech Stack

- **Language**: PHP / Laravel
- **Frontend**: Blade + Alpine.js
- **Database**: MySQL / PostgreSQL
