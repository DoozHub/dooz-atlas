# Tool Comparison

Comparison of AI coding tools used at DoozieSoft.

---

## Quick Comparison

| Tool | Type | Best For | Cost | Learning Curve |
|------|------|----------|------|----------------|
| GitHub Copilot | Inline | Line-by-line completion | Low | Easy |
| Cursor | IDE Agent | Multi-file editing | Medium | Medium |
| Antigravity | Full Agent | Complex workflows | High | Medium |
| Claude Code | Terminal | Shell-based dev | Medium | Medium |
| ChatGPT/Claude Web | Chat | Questions, planning | Variable | Easy |

---

## GitHub Copilot

**Type:** Inline completion

**Pros:**
- Fast, unobtrusive
- Good pattern matching
- Low friction

**Cons:**
- Limited context
- No reasoning
- Can't handle complex tasks

**Best For:** Day-to-day coding, boilerplate

---

## Cursor

**Type:** IDE-integrated agent

**Pros:**
- Good multi-file support
- Familiar IDE environment
- Reasonable cost

**Cons:**
- Can be aggressive with changes
- Context management needed
- Resource intensive

**Best For:** Feature implementation, refactoring

---

## Antigravity

**Type:** Full agentic system

**Pros:**
- Sophisticated planning
- Comprehensive tooling
- Strong guardrails

**Cons:**
- Higher cost
- Requires oversight
- Learning curve

**Best For:** Complex multi-step tasks, large changes

---

## Claude Code

**Type:** Terminal agent

**Pros:**
- Works in terminal
- Good for CLI workflows
- Maintains session context

**Cons:**
- Terminal-only
- Less visual feedback
- Can accumulate costs

**Best For:** Server-side development, scripting

---

## When to Use What

| Task | Recommended Tool |
|------|-----------------|
| Quick completion | Copilot |
| Feature implementation | Cursor |
| Large refactor | Antigravity |
| Server debugging | Claude Code |
| Architecture discussion | Claude/ChatGPT Web |
